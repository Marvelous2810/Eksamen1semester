Css begynder

CSS tager sig af udseende, ved at beskrive hvordan strukturelementer skal se ud.

Webserveren leverer HTML, CSS og Javascript samt billeder mv. til klienter (browsere).

I Netscape version 3 (1996) understøttede CSS (Cascading Style Sheets) og gav nye muligheder for at styre udseendet.

Et reset stylesheet er godt at bruge, når man skal begynde at lave css. Det er et nulstillet stylesheet med samlinger af CSS regler.
Det bruger til at rydde browserens standardformatering af HTML-elementer og fjerne potentielle uoverensstemmelser mellem forskellige browsere

Cascading Style Sheet

Når dit dokument er bygget op med HTML´en er du klar til at designe det med CSS.

Kigger man i et stylesheet ser man på to ting. "selector". selectors kan være mange ting som paragraphs, navne på ideer, headings, classes osv.
Den anden ting er syntax (declaration). Declaration er lavet ud fra "value" par. Der kan være flere declarations i én enkelt regel. Foreksempel
kan en selector som h1 have 2 declarations som værende "font-size" og "text-align". Så har vi altså allerede erklæret to regler til den ene selector.

Margin, Border og padding

Margin er rummet udenfor elementets kant. Det kan justeres med erklæringen (declaration) "margin".
Sætter du margin til auto, finder browseren selv ud af hvor stort marginen på elementet skal være.
Bruger du "px/cm" defineres der med en bestemt målenhed.

Padding er rummet mellem indhold og elementets kant. Dette kan justeres med erklæringen "padding".

Padding kan også defineres med bestemte målenheder.

Kanten for et element kan specificeres med erklæringen "border". Denne erklæring skal indeholde tre værdier (values).

  • width (kantens bredde. Denne kan justeres med værdierne thin, medium, thick eller med en enhed som px/em).
  • Style (kantens design. Denne kan justeres med 10 forskellige værdier, som f.eks. solid, dotted eller dashed).
  • color (kantens farve. Dette kan være en hvilken som helst farve som kanten skal have).
